# decks/

The visual companion to the workshop. Self-contained HTML slide decks — no build step, no dependencies. Open an `.html` in any browser and use **← / →** (or click the dots) to move. Each also has a PDF.

| Deck | Use it with |
|---|---|
| [02-workshop.html](02-workshop.html) | **The Workshop** — a *slideument* covering all six modules end to end: the big why, the concepts, setup and tools, the wiki pattern, the two logs, the evaluation loop, learning with AI, and the capstone. Built to read on its own without a speaker. The slide form of [`workshop/`](../workshop/) as a whole. |
| [05-ai-engineering-v5.html](05-ai-engineering-v5.html) | **Beyond the Answer Machine** — the short talk. Defines *AI* and *critical thinking*, the answer-machine trap, building the system around the model, working a problem with AI, and owning what gets used. The best companion to [`00-orientation`](../workshop/00-orientation.md) and [`01-concepts`](../workshop/01-concepts.md). |
| [01-intro-to-ai-engineering.html](01-intro-to-ai-engineering.html) | **Intro to Applied AI Engineering** — the longer foundation. The abstraction spectrum, the two kinds of intelligence, the ladder (prompt → context → workflow → agent), the agent loop, evaluation as error analysis, and a glossary. Deeper background for [`01-concepts`](../workshop/01-concepts.md). |

The `01` and `05` decks are the slide form of the concepts module. The `02` deck is the slide form of the whole workshop — minimalist styling (white / black / red accent) and written to stand alone as a handout. It ships in two formats:

- **[`02-workshop.html`](02-workshop.html)** — open in any browser; navigate with **← / →**.
- **[`02-workshop.pptx`](02-workshop.pptx)** — a native PowerPoint (21 slides, 16:9) in the deck's own minimalist style (white / black / red). Edit in PowerPoint, Keynote, or Google Slides.
- **[`02-workshop-99p.pptx`](02-workshop-99p.pptx)** — the same content built on the **99p Labs white template**: brand logos, coral accent, Aptos type, and slide numbering. Best opened in PowerPoint (the Aptos font ships with Microsoft Office; other viewers may substitute a fallback font).

**To export the HTML decks as PDF:** open the `.html` in a browser → Print → *Save as PDF*. They're already laid out 16:9 (13.333in × 7.5in), one slide per page.
