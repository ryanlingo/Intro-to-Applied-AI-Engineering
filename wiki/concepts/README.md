# concepts/

Concept and entity pages live here — one page per idea, component, person, or thing the wiki tracks. The agent creates these as it ingests material from [`../../sources/`](../../sources/README.md) and links them from [`../index.md`](../index.md).

This folder starts empty on purpose. It fills with *your* project's knowledge as you go.

Each page should: define the thing, cite the source(s) it came from, link related pages, and mark anything unverified.
