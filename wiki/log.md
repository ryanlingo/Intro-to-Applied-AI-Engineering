# Project Log

The running record of the **work**: what was asked, what the agent did, what changed, decisions made and why, and what is still open. The agent appends here after every working session. Newest entries on top.

## Entry format

```text
## [marker] short title
- Asked:     what you set out to do
- Did:       what the agent actually did
- Changed:   files/pages created or edited
- Decided:   any decision made, and why
- Open:      what is unresolved or to do next
```

Use any marker you like to order entries — a date, a session number, a milestone name. Keep it consistent.

---

## [seed] Repo initialized

- Did: created the schema, workshop modules, sources/ and wiki/ scaffolding.

- Decided: documentation runs on two tracks — this project log and the [learning log](learning-log.md).
- Open: ingest your first real sources (see [workshop module 03](../workshop/03-document-and-wiki.md)).
